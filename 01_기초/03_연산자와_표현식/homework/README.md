# 과제 - 03. 연산자와 표현식

## 문제 1 — 윤년 판별
- 파일명: `Homework01.java`
- 핵심 개념: 비교·논리 연산자, 단락 평가

### 요구사항
- 변수 `year` 에 2024 를 대입합니다.
- 윤년 조건(4의 배수 AND (100의 배수가 아님 OR 400의 배수)) 을 한 표현식으로 작성합니다.
- 결과를 `2024은(는) 윤년? true` 형식으로 출력합니다.

### 예상 출력
```
2024은(는) 윤년? true
```

### 힌트
- `(year % 4 == 0) && (year % 100 != 0 || year % 400 == 0)`

## 문제 2 — 등급 매기기
- 파일명: `Homework02.java`
- 핵심 개념: 삼항 연산자 중첩

### 요구사항
- 점수 `score = 73` 을 선언합니다.
- 삼항 연산자만 사용해 등급(A/B/C/D/F)을 결정합니다.
- `점수=73 -> 등급=C` 형식으로 출력합니다.

### 예상 출력
```
점수=73 -> 등급=C
```

### 힌트
- `(score >= 90) ? "A" : (score >= 80) ? "B" : ...`

## 문제 3 — 비트로 짝홀 판별
- 파일명: `Homework03.java`
- 핵심 개념: 비트 AND `&`

### 요구사항
- 1~5 의 정수에 대해 `(n & 1)` 결과로 짝/홀을 출력합니다.

### 예상 출력
```
1 -> 홀수
2 -> 짝수
3 -> 홀수
4 -> 짝수
5 -> 홀수
```

### 힌트
- `for` 반복은 04편에서 자세히 다루지만, 단순 패턴은 미리 사용해도 좋습니다.
