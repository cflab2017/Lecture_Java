# 과제 - 01. Java 시작하기

## 문제 1 — 자기소개 출력
- 파일명: `Homework01.java`
- 핵심 개념: `public class`, `main`, `System.out.println`

### 요구사항
- 자신의 이름, 좋아하는 언어, 시작 연도를 각각 한 줄씩 출력합니다.
- 출력의 첫 줄은 `=== 자기소개 ===` 입니다.

### 예상 출력
```
=== 자기소개 ===
이름: 홍길동
좋아하는 언어: Java
시작 연도: 2026
```

### 힌트
- `System.out.println` 을 4번 호출하면 됩니다.

## 문제 2 — 명령줄 인사
- 파일명: `Homework02.java`
- 핵심 개념: `args` 배열, 조건 분기

### 요구사항
- 명령줄 인자가 비어 있으면 `사용법: java Homework02 <이름>` 을 출력하고 종료합니다.
- 첫 번째 인자가 주어지면 `안녕하세요, <이름>님!` 을 출력합니다.

### 예상 출력
```
$ java Homework02
사용법: java Homework02 <이름>

$ java Homework02 지수
안녕하세요, 지수님!
```

### 힌트
- `args.length` 로 인자 갯수를 확인합니다.
