# 과제 - 16. 파일 IO

## 문제 1 — TODO 파일 작성·읽기
- 파일명: `Homework01.java`
- 핵심 개념: `Files.write`, `Files.readAllLines`

### 요구사항
- `todo.txt` 파일에 다음 3줄을 씁니다: `우유 사기`, `숙제 끝내기`, `운동 30분`
- 다시 읽어 `1) ...` 형식으로 출력.
- 끝나면 파일을 삭제합니다.

### 예상 출력
```
1) 우유 사기
2) 숙제 끝내기
3) 운동 30분
삭제 완료
```

## 문제 2 — 단어 수 세기
- 파일명: `Homework02.java`
- 핵심 개념: `Files.readString`, `split`

### 요구사항
- `essay.txt` 에 `Java is fun. Java is powerful.` 를 씁니다.
- 공백/구두점 기준으로 단어를 분리해 총 단어 수를 출력합니다.
- 끝나면 파일 삭제.

### 예상 출력
```
단어 수=6
```

### 힌트
- `text.split("[\\s.]+")`
