# 과제 - 17. 문자열 처리

## 문제 1 — 회문 판별
- 파일명: `Homework01.java`
- 핵심 개념: `String` 기본 메서드

### 요구사항
- `static boolean isPalindrome(String s)` — 공백 무시, 대소문자 무시.
- 시험 입력: `"level"`, `"Hello"`, `"A man a plan a canal Panama"`

### 예상 출력
```
level -> true
Hello -> false
A man a plan a canal Panama -> true
```
